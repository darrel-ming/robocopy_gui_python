How to install:
1) Create python virtual environment
python -m venv venv
2) activate the virtual environment
venv\Scripts\activate
3) install dependency packages
cd robocopy
pip install -r requirements.txt
4) click 'robocopy.bat' out of this folder.

:) Good luck!!
